VuPlay Creative LLC is my Roblox development studio that I've been running since 2020. Over the years, I've freelanced for over 80 clients, building everything from small UI tweaks to full game systems.

## What I Build

- Complete game experiences from concept to launch
- UI/UX systems and interfaces
- Gameplay mechanics and scripting in Luau
- Build environments and maps
- Custom tooling and plugins for Roblox Studio

## Workflow

I use Rojo for syncing external code with Roblox Studio, which lets me use proper version control (Git) and work in VS Code while still leveraging Studio's built-in tools for 3D work and testing. This hybrid workflow is pretty standard in professional Roblox development.

## Business Side

Running VuPlay has taught me a lot beyond just coding - client communication, project scoping, invoicing, managing deadlines, and delivering work that meets (or exceeds) expectations. It's been my main entrepreneurial venture through high school and into college.
